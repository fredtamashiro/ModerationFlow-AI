# ModerationFlow AI Frontend

Interface web em Next.js para a base inicial do ModerationFlow AI.

Nesta etapa, o frontend expoe a landing page do novo produto e reaproveita o login administrativo da base tecnica.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Integracao

O frontend consome a API configurada via:

- `NEXT_PUBLIC_API_URL`
- `INTERNAL_API_URL`
